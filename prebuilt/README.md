If you're reading this file on Github while looking for the compiler and debugger binaries, you're in the wrong place! Those binaries are built from source for each new version. This folder only holds the binaries that *aren't*: the files that were compiled once and never need to be compiled again.

For the compiler and debugger binaries, look for "Releases" in the right sidebar on Github. If you want a bleeding-edge version, clone this repository and use "make".
